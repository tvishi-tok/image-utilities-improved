How to use

  1. Add an image by drag-and-drop or the file picker.
  2. Set the target size in KB (try the quick presets).
  3. Enable exact mode if the site is picky about bytes.
  4. Click Compress, then download the result.

FAQs

Can I hit an exact file size?
Use Exact size mode for the closest match; strict portals can still vary slightly by format.

Will quality go down?
Yes, compression trades quality to stay under your KB limit. JPEG and WebP compress more efficiently than PNG.

Is my image uploaded anywhere?
No. All processing stays in your browser on this device.
